# Uber Cab Analysis | Data Analytics Project

## Overview

This project focuses on analyzing Uber ride booking data to identify booking trends, revenue performance, 
ride cancellations, customer ratings, and vehicle-wise operational insights through interactive dashboards and business intelligence reporting.

The project demonstrates an end-to-end Data Analytics workflow using:

* Microsoft Excel
* Power Query
* MySQL
* Power BI

The main objective of this project is to transform raw cab booking data into actionable business insights 
to improve operational efficiency, customer experience, and revenue analysis.

---

# Business Problem

Ride-booking companies often face challenges in:

* Monitoring booking success and cancellation rates
* Analyzing revenue generated by different vehicle types
* Understanding customer ratings and ride performance
* Tracking operational efficiency and travel distance
* Identifying factors affecting customer experience

This project helps solve these challenges through data cleaning, SQL analysis, and interactive dashboard reporting.

# Project Workflow

## 1. Data Loading (MS Excel)

* Imported raw Uber booking dataset into Microsoft Excel
* Reviewed ride booking structure and operational metrics

---

## 2. Data Cleaning (Power Query)

Performed data preprocessing using Power Query:

* Removed duplicate records
* Handled missing values
* Standardized column names
* Corrected inconsistent formatting
* Converted data types for analysis

---

## 3. SQL Analysis (MySQL)

Imported cleaned data into MySQL and performed analytical SQL queries such as:

* Revenue analysis by vehicle type
* Ride cancellation analysis
* Booking success rate analysis
* Distance travelled analysis
* Customer rating analysis
* Vehicle-wise operational performance
